# 추가 선수 한도 확장: n-5 → n-6

## Context
현재 각 코스 그룹에 추가 선수를 최대 5세트(20명)까지 추가할 수 있음.
사용자가 1세트(4명) 더 추가할 수 있도록 6세트(24명)로 확장 요청.

## 수정 파일

### 1. `src/utils/data.js` — addPlayerToCourse 함수
- **236행**: 주석 `최대 20명 = 5세트` → `최대 24명 = 6세트`
- **237행**: 주석 `baseCourse-5` → `baseCourse-6`
- **245행**: `extraCount >= 20` → `extraCount >= 24`

### 2. `src/components/score/ScoreTable.jsx` — 선수 추가 버튼 조건
- **541행**: `extraCount < 20` → `extraCount < 24`

## 검증
- dev 서버에서 한 코스에 선수 추가 버튼을 반복 클릭하여 n-6 (baseCourse-6)까지 생성되는지 확인
- n-6 이후 추가 버튼이 사라지는지 확인
