# 장려상 탭 기능 구현

## 개요
집계 페이지의 장려상 탭을 플레이스홀더에서 실제 기능으로 구현.
개인전 탭(1~5위)에 포함되지 않은 6위부터의 선수를 남녀 분리하여 집계.

## 요구사항
- 칼럼: [남자, 타수, 순위, 여자, 타수]
- 기본 6위~10위 표시 (개인전 1~5위 제외)
- 우측 상단 드롭다운으로 10위~15위까지 범위 선택 가능
- 선택 즉시 화면 반영

## 수정 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/summary/SummaryPage.jsx` | `tournament` prop 전달 (1줄) |
| `src/components/summary/tabs/EncouragementTab.jsx` | 전체 재작성 |

기존 `src/utils/ranking.js`의 `calculateRankings()`, `calculateTotal()` 재사용.

## 구현 내용

### 데이터 처리
```
calculateRankings() → 이름·점수 필터 → 성별 분리 → .slice(5, 5+displayCount)
```
- `maxRank` state (기본 10) 변경 시 useMemo 재계산

### UI 구성
- 순위 선택 드롭다운 (OverviewTab 패턴 재사용)
- 테이블 (IndividualTab과 동일 구조, 행 수만 동적)
