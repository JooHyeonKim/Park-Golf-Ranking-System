# 개인전 탭 기능 구현

## 개요
집계 페이지의 개인전 탭을 플레이스홀더에서 실제 기능으로 구현.
타수가 적은 순으로 남녀 각 상위 5명(우승~5위)을 집계하여 표시.

## 요구사항
- 칼럼: [남자, 타수, 순위, 여자, 타수]
- 순위: 우승, 준우승, 3위, 4위, 5위
- 타수 오름차순 정렬 (낮을수록 상위)
- 동점 시 D→C→B→A 코스 순으로 비교

## 수정 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/summary/tabs/IndividualTab.jsx` | 전체 재작성 |

기존 `src/utils/ranking.js`의 `calculateRankings()`, `calculateTotal()` 재사용.

## 구현 내용

### 데이터 처리
```
calculateRankings(tournament.players)
  → filter(이름·점수 있는 선수)
  → 성별 분리 (남/여)
  → .slice(0, 5)  // 상위 5명
```
- `useMemo`로 `tournament.players` 변경 시 재계산

### UI 구성
- 5행 고정 테이블 (우승~5위)
- 남자: 파란색 헤더 (`bg-blue-200` / `bg-blue-100`)
- 여자: 분홍색 헤더 (`bg-pink-200` / `bg-pink-100`)
- 순위: 회색 중앙 칼럼 (`bg-gray-300` / `bg-gray-200`)
- 교대 행 색상 (`bg-white` / `bg-gray-50`)

### 테이블 레이아웃
```
┌──────────┬──────┬────────┬──────────┬──────┐
│ 남자     │ 타수 │ 순위   │ 여자     │ 타수 │
├──────────┼──────┼────────┼──────────┼──────┤
│ 성명     │ 타수 │        │ 성명     │ 타수 │
├──────────┼──────┼────────┼──────────┼──────┤
│          │      │ 우승   │          │      │
│          │      │ 준우승 │          │      │
│          │      │ 3위    │          │      │
│          │      │ 4위    │          │      │
│          │      │ 5위    │          │      │
└──────────┴──────┴────────┴──────────┴──────┘
```
