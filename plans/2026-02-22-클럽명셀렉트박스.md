# 클럽명 입력방식 변경: 텍스트 입력 → 셀렉트 박스

## Context

현재 점수 입력 화면(ScoreTable)에서 클럽명을 자유 텍스트로 입력하고 있어 오타/불일치 문제가 발생할 수 있음. 고정된 13개 클럽 목록에서 선택하는 방식으로 변경하여 데이터 일관성을 확보하고, 단체전(TeamTab) 클럽별 그룹핑의 정확도를 높임.

## 고정 클럽 목록

방배, 양재, 잠원, 반포, 서리풀, 서래, 우면, 서초, 내곡, 청계, 이수, 매헌, 향목

## 수정 대상 파일 (2개)

### 1. `src/utils/data.js` — 클럽 상수 추가

파일 상단(line 1 아래)에 `CLUBS` 배열 export 추가:

```javascript
export const CLUBS = [
  '방배', '양재', '잠원', '반포', '서리풀',
  '서래', '우면', '서초', '내곡', '청계',
  '이수', '매헌', '향목'
];
```

### 2. `src/components/score/ScoreTable.jsx` — 3곳 수정

**a) import 추가 (line 2 부근)**

```javascript
import { CLUBS } from '../../utils/data';
```

**b) 클럽 텍스트 input → select 변경 (lines 224-233)**

기존 `<input type="text">` 를 성별 select(lines 212-221)와 동일한 패턴의 `<select>`로 교체:

```jsx
{/* 클럽 */}
<td className="py-2 px-2 border-r">
  <select
    value={player.club || ''}
    onChange={(e) => handleInputChange(player.id, 'club', e.target.value)}
    className="w-full px-2 py-1 border rounded focus:outline-none focus:ring-1 focus:ring-green-500"
  >
    <option value="">-</option>
    {CLUBS.map((club) => (
      <option key={club} value={club}>{club}</option>
    ))}
  </select>
</td>
```

**c) 테스트 데이터 클럽 배열 업데이트 (line 56)**

```diff
- const clubs = ['그린파크', '레이크힐', '선샤인', '블루밸리', '골든게이트'];
+ const clubs = CLUBS;
```

## 수정하지 않는 파일

| 파일 | 이유 |
|------|------|
| `src/components/summary/tabs/TeamTab.jsx` | `player.club?.trim()`으로 그룹핑 — 고정 목록 사용 시 더 안정적으로 동작, 변경 불필요 |
| `src/components/summary/tabs/OverviewTab.jsx` | `{player.club \|\| '-'}` 읽기 전용 표시 — 변경 불필요 |
| `src/utils/data.js` (player 모델) | `club: ''` 기본값은 select의 빈 option과 매칭되므로 변경 불필요 |

## 검증 방법

1. `npm run dev`로 개발 서버 실행
2. 대회 생성 → 점수 입력 화면 진입
3. 클럽 컬럼이 셀렉트 박스로 표시되는지 확인
4. 13개 클럽 목록이 정확히 나오는지 확인
5. 클럽 선택 후 결과보기 → 개요 탭에서 선택한 클럽명 표시 확인
6. 단체전 탭에서 클럽별 그룹핑 정상 동작 확인
7. 테스트 데이터 생성 버튼 → 새 클럽 목록에서 랜덤 배정 확인
