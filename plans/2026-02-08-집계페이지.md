# 집계 페이지 구현 계획

## 개요
대회 점수 입력 후 집계 결과를 확인할 수 있는 탭 기반 페이지 추가

## 탭 구성
| 탭 | 이름 | 내용 |
|----|------|------|
| 1 | 전체 현황 | ScoreTable과 동일한 테이블 (입력창 없이 텍스트만) |
| 2 | 개인전 | 추후 구현 (플레이스홀더) |
| 3 | 장려상 | 추후 구현 (플레이스홀더) |

## 구현 파일

### 신규 생성
```
src/components/summary/
├── SummaryPage.jsx          # 메인 페이지 (탭 네비게이션 포함)
└── tabs/
    ├── OverviewTab.jsx      # 전체 현황 탭
    ├── IndividualTab.jsx    # 개인전 탭 (플레이스홀더)
    └── EncouragementTab.jsx # 장려상 탭 (플레이스홀더)
```

### 수정
- `src/App.jsx` - screenMode 상태 추가, 라우팅 확장
- `src/components/score/ScoreTable.jsx` - 집계 보기 버튼 추가

## 화면 흐름
```
TournamentList → ScoreTable → SummaryPage
     ↑              ↑             ↓
     └──────────────┴─────────────┘
```
